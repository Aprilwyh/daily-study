### px、em、rem
- px：绝对长度。相对于显示器屏幕分辨率而言。设置之后无法随着页面大小改变而改变
- em：相对长度。值不固定，会继承父级元素的字体大小，一般都是以<body>的“font-size”为基准。
```html
<div>
    <p></p>
</div>
```
```css
div {
    font-size: 40px;
    width: 7.5em; /* 300px */
}
p {
    font-size: 0.5em; /* 20px */
    width: 7.5em; /* 150px */
}
```
- rem：相对长度。相对根元素（<html>）
```html
<div>
    <p></p>
</div>
```
```css
html {
    font-size: 10px;
}
div {
    font-size: 4rem; /* 40px */
    width: 20rem; /* 200px */
}
p {
    font-size: 2rem; /* 20px */
    width: 10rem; /* 100px */
}
```
